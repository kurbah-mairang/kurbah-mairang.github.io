About html



HTML is a standard markup language used for designing website.Website is a collection of webpages.HTML stands for Hyper Text Markup Language. Html describe the structure of a webpage.It provides to us a series of elements where we can use to design a webpage.These elements are represented by tags. Browser does not display the tags but instead used them to render the content of the webpage.The following list shows some of the tags provided to us by HTML:

There are two types of tags in HTML:

container tags:
These types of tags consists of a starting as well as an ending tag.
empty tags:
These types of tags consists of a staring tag only

List of tags in HTML:

  • <p>: This tag defines a paragraph.It is a container tag.
  • <br>: It breaks the currrnt line and start showing text in the next line.
  • <body>: This tags describe how our webpage would be displayed.
  • <img>: This tag includes an image in our webpage.
  • The heading tags :This tag is used to include a heading in our web page.
  • HTML links : HTML links are defined with the <a> tag.
  • <hr>: This tag is used to draw a horizontal line across our webpage.

HTML also provides us various attributes to make our webpages more attractable
